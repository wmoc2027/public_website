# WMG/WMOC2027 オリエンテーリング・リハーサル2日間大会

## はじめに

2027年5月、30歳以上*1であれば誰でも参加できる生涯スポーツの国際大会「ワールドマスターズゲームズ（WMG）」が関西圏で開催されます。公式競技として35競技59種目があり、オリエンテーリング競技もその一つですが、本大会は「世界マスターズオリエンテーリング選手権（WMOC）」としても開催されます。

WMOC2027は、アジア初の開催として海外から1,000人以上の参加者が集まることが期待されています。

競技には都市部や公園で行われるスプリント種目と、森林で行われるフォレスト種目があります。兵庫県北部に位置する香美町、養父市、神河町がこれらの種目の開催自治体となります。

大会の1年前、世界レベルの大会を円滑に運営し、機運を高めるため、開催市町の実行委員会と連携してリハーサル大会を開催いたします。

*1 オリエンテーリング競技は35歳以上が対象 ↩︎
