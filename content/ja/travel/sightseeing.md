WMOC開催地である[兵庫県](https://www.hyogo-tourism.jp/)には多くの観光名所があります。

{{< card-group padding="3" gutter="2" cols="1">}}
{{< card title="Kami Town" icon="flag kami fa-8x" >}}
    庫県北部に位置する[香美町](https://www.town.mikata-kami.lg.jp/www/index.html)は、海、山、川など豊かな自然に囲まれた地域です。山陰海岸ユネスコ世界ジオパークの構成地域であり、貴重な地質遺産を数多く有しています。また、神戸牛の原種牛である但馬牛の産地としても知られています。
{{< /card >}}
{{< card title="香住地区" >}}
    香住地区は日本海に面しており、その美しい景観は息を呑むほどです。夏には多くの海水浴客で賑わい、冬には漁師たちがズワイガニを水揚げし、周辺の宿泊施設ではカニ料理を堪能できます。
{{< image src="image/kasumi_seacoast.jpg" caption="Kasumi seacoast" >}}
{{< /card >}}
{{< card title="Kamikawa town" icon="flag kamikawa fa-8x" >}}
    [神河町](https://www.town.kamikawa.hyogo.jp/)は、映画やドラマのロケ地として有名な砥峰高原と峰山高原を有しています。また、清流が流れる越智川や1,000メートル級の山々など、豊かな自然に囲まれています。冬は雪景色、秋はススキ、夏はホタル、春は桜など、美しい景色を堪能できます。
{{< image src="image/mineyama_plateau.jpg" caption="Autumn in Tonomine plateau" >}}
{{< /card >}}
{{< card title="Yabu City" icon="flag yabu fa-8x" >}}
    兵庫県北部、但馬地方に位置する政令指定都市、[養父市](https://www.city.yabu.hyogo.jp/index.html)。2004年に養父市、大谷町、八鹿町、関宮町の4町が合併して誕生しました。山あいの田園地帯で、川が流れ、冬には雪景色が広がります。この地域は、アウトドアアクティビティ、神聖な祭りや文化的な祭り、美しい天滝や町を見下ろす雄大な氷ノ山などの自然スポットで知られています。養父市は四季折々の緑豊かな美しい自然が広がっています。日本はハイテクや未来志向の街として知られていますが、養父市では昔ながらの田舎の雰囲気を味わうことができます。パラグライダー、キャニオニング、登山、スキー、キャンプなど、アウトドア愛好家には最適な場所です。タイムスリップしたかのような、日本の伝統的な景観をお楽しみください。
{{< image src="image/bekkutanada.jpg" caption="The \"Tanada\" at Bekku area" >}}
{{< /card >}}
