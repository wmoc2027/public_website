    {{< card title="WMG2027への登録 / WMG マスターズビレッジ" icon="fas 1" >}}
WMG2027の参加者認証はオンラインで行われます。WMG参加者全員の配布物と交通パスは各WMG開催都市のマスターズビレッジ（地図上の赤色塗りつぶし丸の場所）で受け取ることができます。
> [!NOTE]
> マスターズビレッジの設置場所は現在未定です。
    {{< /card >}}
    {{< card title="レースナンバーの受け取り / WMOC イベントセンター" icon="fas 2" >}}
WMOCイベントセンターは、[兵庫県香美町の香住B&G海洋センター](https://maps.app.goo.gl/DgHjt1WfHyfLfLCx5)（地図上の中抜き赤丸の場所）です。
WMOCオリエンテーリングの参加受付（レースナンバー等の受け取り）は、2027年5月20日と21日にイベントセンターで行われます。また、その後各オリエンテーリング競技場でもインフォメーションデスクが設置され、参加受付を行うことができます。
    {{< /card >}}
    {{< card title="登録した競技に参加する" icon="fas 3" >}}
イベントセンターでナンバーカード等を受け取ったらあらかじめ本WEBサイトで公開されたスタート時刻で競技スタートします。時間に余裕を持って十分にウォーミングアップを行った上でスタート地区へお越しください。
    {{< /card >}}
